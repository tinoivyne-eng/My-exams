2) Linear Search

This checks items one by one.
Example:

You look at the first item, then the second, then the third.
It is slow if there are many items. 

3)Binary Search

This works by cutting the list in half each time.

Example:

-Start in the middle
-Go left or right
-Keep cutting in half

It is very fast
